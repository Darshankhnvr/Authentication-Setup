# Full Stack Authentication App

Simple full stack authentication system with Login & Signup.

## Tech Stack
- React.js
- Tailwind CSS
- shadcn/ui
- Context API
- Node.js
- Express
- Axios

## Features
- Signup & Login
- Form validations
- Password hashing
- Global auth state
- Clean UI

## Setup

### Client
cd client  
npm install  
npm run dev  

### Server
cd server  
npm install  
npm run dev
